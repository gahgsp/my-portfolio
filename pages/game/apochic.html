Apochic

Apochic is a 2D narrative-driven game where your main objective is to unveil what happened to the world and to the human race.
This game was the first game I made focusing on narrative to practice my storytelling skills.

Roles:  Game Designer, Game Developer

Tools:  Unity

Platform:  PC

What I did: 

  1. Designed-developed the entire game from scratch;
  2. Created a world and a background story to build the game mechanics;
  3. Built a customizable dialog system to use throughout the game.

What I learned: 

  1. Optimized usage of Tilemaps in Unity;
  2. Game development focused on narrative and storytelling.
