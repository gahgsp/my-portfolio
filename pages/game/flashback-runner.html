Flashback Runner

Flashback Runner is an infinite runner with the entire map being procedurally generated during gameplay.
The game is set in the 80's and replicates a vintage athmosphere using color palettes and songs that refer to the 80's.

Roles:  Game Designer, Game Developer

Tools:  Unity

Platform:  PC

What I did: 

  1. Designed-developed the entire game from scratch;
  2. Use of SFX and background music to create an immersive experience;
  3. Use of thematic color palettes and 3D models to create an environment with references to the 80's.

What I learned: 

  1. Procedural generation;
  2. Events Delegation in C# to communicate between scripts;
  3. Post-Processing effects with Unity built-in tool;
  4. Responsive UI to allow multiple resolutions.
