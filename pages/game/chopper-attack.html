Chopper Attack

Chopper Attack is a 2D game developed using a handmade game engine using C++ and SDL2.

The goal of this project is to learn more in depth about C++ and game engines. Also, this is still a WIP because as I progress with my C++ studies, I want to apply all the best practices and code improvements into the engine.

Roles:  Game Developer

Tools:  C++, SDL2, Lua

Team:  Solo

Platform:  PC

What I did: 

  1. Developed the engine from scratch using C++ and SDL2 with the help of courses and internet tutorials;
  2. Built a scalable and easy to maintain ECS (Entity Component System) in the engine;
  3. Implemented support to read and import entities from Lua files.

What I learned: 

  1. How to develop and how game engines work;
  2. Improved my C++ skills towards game development, using external libraries such as SDL2;
  3. How to embed code from other programming languages (Lua) into an engine developed in C++.
