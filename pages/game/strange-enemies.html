A lot of... Strange Enemies!

This game is a 2D platformer with action elements that was developed in under 48 hours for the Quarantine Jam with the theme "A lof of everything you want!".
This was my first jam and I had the opportunity to work with another experienced game programmer.
I was very fortunate to have this experience as I learned different subjects on game development.

Roles:  Game Designer, Game Developer, Level Designer

Tools:  Unity

Team:  2 members

Platform:  PC

What I did: 

  1. Designed the game core mechanics;
  2. Designed the sole level of the game, using different approaches to diversify the platforms mechanics;
  3. Implemented the player actions and movement.

What I learned: 

  1. Create and develop a game in less than 48 hours under pressure;
  2. Tasks organization to keep the objectives aligned with all the team members;
  3. Object Pooling to keep the game with high performance even when spawning many entities.
