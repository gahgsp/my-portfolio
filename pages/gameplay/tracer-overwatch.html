Tracer - Overwatch

The goal of this project was to recreate Tracer's core mechanics in order to improve my programming skills using Unity and C#.

Tools:  Unity, C#

Development process: 

Before starting programming the mechanics, I firstly decided what would be the scope for this project.
As I wanted to learn more about the programming side of Unity, I've decided to focus on the core mechanics and it's functional side, excluding many of the visual effects while casting the abilities.
With this reduced scope I was able to focus solely on developing a highly customizable and clean code, that can be applied in different projects.

What I did: 

  1. Developed from scratch the core mechanics and abilities: Pulse Pistols, Blink, Recall and Pulse Bomb;
  2. Implemented the First Person features e.g. Camera and Movement;
  3. Built a high customizable and easy to maintain code that can be used in many different projects.
