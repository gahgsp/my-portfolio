AI Unreal Engine Course - Final Project

This is the result after completing the "Introduction to AI with Blueprints" available on Learn Unreal.
After finishing the course, I developed new features to the project as a challenge to improve the AI character.

Tools:  Unreal Engine, Blueprint, EQS, Blackboard, Behavior Tree

AI features: 

The AI consists in the following flow:

The AI character starts searching for the player on a random path, each step the AI character takes, a "Hunger Level" is increased. If the AI character finds the player, it starts to follow the player and once it gets closer enough, the AI character will start attacking the player.
Chasing the player also increases the "Hunger Level" that when it reaches the maximum level, the AI character will hide from the player and afterwards it will start looking for "Food Sources" using the EQS feature.

The AI can also hear the player. If the AI character hears something, it will suspect and then investigate.
The AI character has the ability to jump with the use of Navigation Links.

What I've learned: 

  1. Basics features for AI implementation on Unreal Engine such as the Blackboard and Behavior Trees;
  2. How to implement interactions between the Blueprints and the AI features like AI Controller, NavMesh, AI Perception and AI Gameplay Debugger;
  3. How and when to use Sequences and Selectors;
  4. Use of EQS system in order to make the AI interact with the environment;
  5. Implementation of Tasks, Services and Decorators.
