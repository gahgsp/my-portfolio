Metal Gear Solid: The Phantom Pain - Int Scope

The goal of this project was to recreate the basic features on the Int Scope from the game Metal Gear Solid: The Phantom Pain in order to learn more about Unreal Engine plus the UMG.

Tools:  Unreal Engine, Blueprint

Development process: 

For this project, once I decided that I would recreate this item from the game, I had already in mind the core mechanics I would like to have before calling the final project:

  • This project should have the same camera behavior as the game: it should be a third person camera but once the Int Scope is equipped, the camera should smoothly switch to first person;
  • It should be possible to tag an enemy and afterwards, it could be seen through walls and other objects;
  • A tag icon should appear next to the enemy, with the distance from the player to the enemy being calculated in real time;
  • The zoom counter on the bottom left should be updated as the player keeps zooming in.

During the development of this project, I really struggled to make the enemy be seen through the objects. Fortunately, I've found this awesome tutorial by Rodrigo Villani that teaches in details how to work with the material system on the Unreal Engine.

What I learned: 

  1. Improved my skills using the Blueprints from the Unreal Engine;
  2. Gained some knowledge working with the UMG to create interfaces that have dynamic values and effects;
  3. Learned how to create complex materials using the Unreal Engine built-in tool.
